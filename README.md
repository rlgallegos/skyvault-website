## Introduction

**Skyvault** is an ongoing project designed to market the Skyvault mobile app (which is currently available for both iOS and Android).

## Technologies

The project leverages the following technologies and programming languages:

- **Frontend**: JavaScript, React.js
- **Libraries**: react-scroll, react-icons
- **Other Dependencies**: Default React dependencies

## Key Features

Skyvault offers the following key features and functionalities:

- **Showcasing Skyvault App**: It serves as a platform to showcase the capabilities and features of the Skyvault mobile app.
- **Tips and Tricks**: Provides users with tips and tricks on how to use the Skyvault app effectively.
- **Privacy Disclosures**: Offers access to important privacy disclosures and policies.
- **Contact**: Allows users to get in touch with the Skyvault company.

## Usage

Users can easily navigate and interact with the Skyvault Project through a user-friendly interface. A Navbar, which can be a dropdown Navbar on mobile devices, follows users as they scroll, enabling smooth transitions between different sections of the page.

## Installation

To run the project locally, ensure you have the default React dependencies installed. Additional project-specific dependencies are listed in the project.json file.
